# AI & Remote Work — Platform Reference

You pivoted away from forex-only searches. This page covers the remote/AI
landscape and how to get value from each source with CareerOps.

Think of these in three buckets:

1. **Sign-up platforms** — not job boards, not scrapable. You apply once, pass their vetting, then they send you work.
2. **Auto-scannable aggregators** — CareerOps pulls from these automatically.
3. **Browse-and-paste sites** — use **New → Paste URL** on individual listings.

## 2. Auto-scannable aggregators (CareerOps pulls these)

### RemoteOK — already seeded

Five RemoteOK tag filters are in your portals list by default after the latest update:

- `RemoteOK · AI` — tags: ai, ml, llm, ai-engineer
- `RemoteOK · Marketing` — content, copy, SEO, marketing
- `RemoteOK · Design` — UX, UI, product-design
- `RemoteOK · Finance & Crypto` — finance, fintech, crypto, trading
- `RemoteOK · Non-tech` — customer support, ops, community, HR, recruiting

Run a scan and you'll get thousands of remote jobs across these areas. Then filter the **Jobs** page by title or description keyword and evaluate the ones that fit.

### Add your own RemoteOK filters

In the **Scan** page, click **Add**:
- ATS: `RemoteOK (tags)`
- Board slug: comma-separated tags. Use `all` to get every remote job, or specific tags like `writing,editor,blog`.

### Other auto-scan targets (already supported)

- Greenhouse: any `boards.greenhouse.io/{slug}`
- Ashby: any `jobs.ashbyhq.com/{slug}`
- Lever: any `jobs.lever.co/{slug}`
- Workable: any `apply.workable.com/{slug}`
- SmartRecruiters: any `jobs.smartrecruiters.com/{slug}`

**LinkedIn Jobs** is the biggest source in the world but actively blocks scraping. Workflow: browse LinkedIn, find a role, copy the URL, paste into **New → Paste manually** (not Paste URL — LinkedIn blocks the fetch). Copy the description text too. ~1 minute per job, scales well.
